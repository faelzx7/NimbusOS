# marketing/

Onde o sistema salva o que produz pro negócio em si: carrosséis, artigos de blog, campanhas de
anúncio, respostas de avaliação, relatórios de SEO. Organizado por skill (ex: `marketing/conteudo/`,
`marketing/campanhas/`, `marketing/seo/`) — cada skill cria a subpasta que precisar na primeira vez
que rodar.
